Create S2S EMI Payment

This page demonstrates Server-to-Server (S2S) EMI payment integration for both cardless EMI and card EMI (PayU). For cardless EMI: select provider and tenure; you will be redirected to the lender’s page. For card EMI (PayU): use bank code and card details.
